# Aeroscore
AeroScore: The score for aircraft predictive maintenance and performance metrics in one number to optimize leasing, buying, and repairs.

AeroScore is a smart, credit-score-style rating system for aircraft assets. It combines real-time performance metrics (maintenance compliance, utilization & wear, dispatch reliability, fuel efficiency, incident history) with advanced predictive maintenance algorithms (RUL estimation, degradation trends, AHM alerts, and service-delay probability) into a single, easy-to-understand 300-850 score.This helps lessors, airlines, and MRO teams instantly decide which aircraft is ready for the next lease, purchase, heavy maintenance, or retirement while clearly quantifying the uncertainty of return-to-service timelines.
